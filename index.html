1. Physically based modeling

Constrained dynamics simulation is inspired by a Carnegie Mellon University's course.

The main feature of the simulation is that it's using ideal constraints as joints between material points, not springs.
The simulation uses the explicit fourth-order Runge-Kutta method in order to solve an ordinary differential equation of the simulation state in the phase space.
If you're keen on the math model the simulation based on, please take a look at 'Constrained Dynamics' lecture.
